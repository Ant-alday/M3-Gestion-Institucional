# M3-Gestion-Institucional
Administra la estructura organizacional del Ayuntamiento.  Se encarga de mantener el catálogo de departamentos del ayuntamiento y el registro del personal operativo. El Microservicio 2 de Incidencias consulta este servicio cada vez que necesita asignar a alguien para atender un reporte, y también cuando lo libera una vez que se resuelve.
